# baize-seo-tool

## 本地运行

在根目录下建 .env 文件

> 配置格式

```
SITE_URL=https://baize.plume.vip
SITEMAP_URL=https://baize.plume.vip/sitemap.xml
BAIDU_TOKEN=你的token
HOME_PAGE_URL=https://baize.plume.vip
```

## 部署到vercel
